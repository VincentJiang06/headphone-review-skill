# headphone-review-skill

[English Version](./README.md)

`headphone-review` 是一个用于输出买家视角耳机评测的 Codex skill，适用于耳机、IEM、耳塞和 TWS 产品。

它的目标不是机械搬运参数表，而是回答真正和购买有关的问题：

- 这款耳机大致是什么声音
- 它更适合什么人
- 它不适合什么人
- 它真正的强项和代价分别是什么

## 当前版本

- `1.0.1`

## 这个 Skill 会做什么

在开始写评测之前，这个 skill 会先询问两件事：

1. 你想要中文还是英文输出
2. 你想要更客观的描述，还是更情绪化、抽象一些的描述

确认后，它会：

- 锁定具体型号、代际与形态
- 搜集官方信息和多语言评测共识
- 区分已确认事实与跨来源共识
- 把声音特征翻译成购买建议
- 输出纯 Markdown 的短文式评测

## 输出特点

- 开头先给出语言、描述风格、一句话短评、风格关键词和置信度
- 正文保持文章体，而不是 bullet 堆砌
- 优先保障中文输出质量
- 对型号歧义和证据不足会明确标注假设和低置信度

## 适用场景

- 快速判断某款耳机是否适合自己
- 把零散的多语言评测整理成一份买家视角结论
- 测试中文短文式音频评测输出
- 对多个型号进行调音与人群定位比较

## 使用方式

在 Codex 中调用：

```text
$headphone-review
```

然后输入型号。skill 会先询问语言和描述风格，再继续生成评测。
